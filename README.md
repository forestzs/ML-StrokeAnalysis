# ML-StrokeAnalysis

This repository contains baseline ML scripts and a cleaned stroke dataset for a simple tabular classification/regression analysis project.

## Contents

- `code/`
  - `decision_tree.py`
  - `linear.py`
  - `logic.py`
  - `poly.py`
- `data_clean/`
  - stroke_data.csv (CSV)
- `paper/`
  - Feature Impact Analysis and Machine Learning-based Stroke Probability Prediction.pdf

## Notes

- This repo is intended for educational and reproducible experiments.
- Dataset and scripts are kept lightweight; no large training artifacts are included.
- This paper was accepted by CONF-CDS 2023 but was not published in the proceedings (camera-ready not submitted / registration not completed). No copyright transfer was signed; the authors retain the copyright.
